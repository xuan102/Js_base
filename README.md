### :wrench: Start with terminal
in your terminal tape:
- git clone https://github.com/DoubleLama/shopkitties.git
- bundle install
- rails db:create
- rails db:migrate
- rails db:seed

Now you can visit it on https://localhost:3000/

### It is built by
- Ruby on rails 5.2 (backend)
- Boostrap (front)
